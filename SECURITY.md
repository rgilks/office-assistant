# Security Policy

## Supported Versions

This project is currently maintained on the `main` branch.

## Reporting a Vulnerability

Please do **not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities using GitHub's private vulnerability reporting flow:
1. Open the repository on GitHub
2. Go to **Security** -> **Advisories**
3. Click **Report a vulnerability**

If possible, include:
- A clear description of the issue and impact
- Steps to reproduce
- Any suggested remediation

Acknowledgment target: within 3 business days.
